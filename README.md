# 📊 Sales Performance Analysis Dashboard (Python + Power BI)

## 📌 Overview
This project analyzes sales data to identify trends, top-performing products, and regional performance.

## 🎯 Problem Statement
The goal of this project is to analyze sales data to understand revenue trends, product performance, and regional sales distribution in order to support better business decisions.

## 🛠 Tools Used
- Python (Pandas)
- Power BI
- Excel

## 🔍 Key Insights
- The East region generated the highest total revenue.
- Keyboards are the best-performing product category.
- Phones show lower sales performance across most regions.
- Sales distribution varies significantly by region.

## 📌 Conclusion
This analysis helps identify strong and weak product areas, enabling better business decision-making for improving sales performance.
